AJAX入门

AJAX的定义

Asynchronous Javascript and XML 的缩写。
最早出现在 2005 年的 Google Suggest，是在浏览器端进行网络编程(发送请求、接收响应)的技术方案，它使我们可以通过 JavaScript 直接获取服务端最新的内容而不必重新加载页面。让 Web 更能接近桌面应用的用户体验。

在此之前，解决方案是？

以下可以让浏览器发出对于服务端的请求，获得服务端的数据：

  • 地址栏输入地址，回车然后刷新
  • 特定元素的href或者src属性
  • 表单提交

有了AJAX，就可以直接通过JavaScript直接发送网络请求。

AJAX到底是什么

AJAX实质上是浏览器提供的一套API（不知道什么是API？？点这里）,可以通过Javascript调用，从而实现通过代码控制请求与响应，实现了网络编程。

AJAX快速上手

第几行 实质 注意
第一行 的 new 创建了一个XMLHttpRequest类型的对象 相当于打开了一个浏览器
第二行的 xhr.open() 使用了GET方法对于浏览器提出请求，打开了一个连接 相当于在地址栏输入访问地址
第三行的send 通过刚刚的连接发送了请求 类似于按了回车键
第四行的onreadystatechange 这是一个制定xhr状态变化事件处理函数 类似与网页呈现后的操作
第五行的readystate 通过这个是否是4来判断本次请求的响应是否接收完成 这个值最初是0
第六行的this 就是指通过xhr的responseText获取到的响应的响应体 readystate为3的时候，响应体正在下载，有可能为空也有可能是不完整，在readystate为3的时候处理响应体不保险

Insert a Pic Into Blog

发表于 2018-09-08

How to insert a pic into Blog builded by HEXO?

First

  1. find your username.github.io folder
  2. find your file — _config.yml
  3. change the attribute of post_asset_folder from false to true
  4. then you should run
    npm install hexo-asset-image --save in terminal under your folder username.github.io folder
    there is a message:
    ╭─────────────────────────────────────╮
    `Update available 5.6.0 → 6.4.1`
`Run npm i -g npm to update`
    ╰─────────────────────────────────────╯
  5. after download done, try to run hexo s to test your blog.
